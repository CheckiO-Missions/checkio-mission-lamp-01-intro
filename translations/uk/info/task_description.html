Цією місією я хочу почати серію місій з лампочками. Вони допоможуть вам зрозуміти концепцію процесів і оцінку ефективності процесів. Замість лампочок у реальному житті може стояти техніка, ефективність якої треба розраховувати, або працівники, які ходять на роботу, і їм треба розраховувати зарплату.

Перша місія досить проста. Є лампочка, яка за замовчуванням вимкнена, і кнопка, при натисканні на яку лампочка перемикає свій стан. Це означає, що якщо лампочка вимкнена і натиснута кнопка, світло вмикається, а якщо натиснути ще раз, то вимикається.

(Все легко. Я впевнений, що якщо ти потрапив на цю місію, ти маєш це зрозуміти, але про всяк випадок я додаю візуальне зображення.)

Вхід функції - це {% if is_js %}масив з Date об'єктами{% else %}список з datetime об'єктами{% endif %} - це дата і час натискання кнопки. Твоє завдання визначити, скільки часу горить лампочка.

Вхідні дані: {% if is_js %}Масив з Date об'єктами{% else %}Список з datetime об'єктами{% endif %}.

Вихідні дані: Секунди роботи лампочки (ціле число).

Передумова:

  • {% if is_js %}Масив{% else %}Список{% endif %} натискання кнопки завжди сортується за зростанням
  • {% if is_js %}Масив{% else %}Список{% endif %} натискання кнопки не має повторюваних елементів (тобто результат завжди повинен бути більше 0)
  • Кількість елементів завжди парна (світло з часом вимикається)
  • Мінімальна можлива дата - 01.01.1970
  • Максимальна можлива дата 9999-12-31
